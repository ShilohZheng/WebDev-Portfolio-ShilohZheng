Implemented Accessibility Features

I designed this website with accessibility in mind, following beginner-level WCAG guidelines to help all possible users have a better experience.

  • Semantic HTML5 tags:
    // HTML: Helps screen readers understand structure. 
    <main> <section> <footer>
  • Color contrast:
    // CSS: Improves readability. 
    color: #333;
    background: #f6f6f6;
  • Keyboard-navigable links:
    // HTML: All nav links use <a> tags. 
    <nav><a href="...">Link</a></nav>
  • Readable fonts:
    // CSS: Uses readable font and size. 
    font-family: 'Times New Roman';
    font-size: 16px;
  • Responsive layout:
    // CSS media query. 
    @media (max-width: 768px) {...}

Images

All images use alt text.
// HTML: Screen reader access. 

<img src="..." alt="Corgi smiling">

My Goal

Accessibility matters because everyone deserves to access information equally. This project helped me understand how even small changes can make a big difference.
